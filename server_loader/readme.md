# 中文說明
開服只執行一次的cfg檔案

* 原理
    * 啟動伺服器之後自動執行一個cfg檔案，只會執行一次

* 用意在哪?
	* 有些指令只需要一開始設定一次就行了，不適合寫在```cfg/server.cfg```，譬如伺服器人數上限、遊戲模式等等
        * ```cfg/server.cfg``` 是每次換地圖必定會執行的檔案

* 伺服器只執行一次的檔案
    * cfg/server_startup.cfg

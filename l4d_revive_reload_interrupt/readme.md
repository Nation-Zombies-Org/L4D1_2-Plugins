# 中文說明
解決裝子彈的時候拯救隊友會卡彈的問題 (存在超過十年的Bug)

* 原理
	* 修復存在超過十年的智障Bug，會讓你的武器卡彈

* 如何重現Bug
    1. 把武器彈夾打空然後裝子彈
    2. 裝子彈時拯救倒地的隊友
    3. 停止拯救倒地的隊友
    4. 嘗試左鍵開槍 (卡彈 / 啞彈 / 重複裝彈)

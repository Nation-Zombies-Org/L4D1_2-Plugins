# 中文說明
車子爆炸啦!

* 原理
	* 地圖上可以移動的車子如果受到一定程度的傷害，將會爆炸，並震飛周圍的倖存者
    * 地圖上某些車子不能移動也就不能爆炸，認真你就輸了

* 功能
	* 可設置爆炸所產生的傷害
	* 可設置爆炸範圍
	* 可設置車子爆炸之後過一段時間自動移除 (避免擋住地圖路線)
	* 可設置爆炸威力 (倖存者飛得越遠)
	* 爆炸是否引發屍潮與機率
	* 可設置特定的地圖開關

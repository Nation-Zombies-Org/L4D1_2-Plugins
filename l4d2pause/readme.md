# 中文說明
管理員可以強制暫停遊戲，也只有管理員能解除暫停

* 原理
	* 管理員輸入 !forcepause 強制暫停遊戲
	* 管理員輸入 !forcepause 解除暫停
	* 當發動暫停的管理員離開伺服器時，遊戲會自動解除暫停

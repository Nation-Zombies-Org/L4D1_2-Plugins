# 中文說明
Hunter的高撲造成核彈衝擊波

* 原理
	* AI Hunter或者真人Hunter的高撲如果超過一定高度，則造成核彈衝擊波
        * 給予周圍玩家額外傷害
        * 震飛周圍玩家

* 功能
    * 可設置核彈衝擊範圍與額外傷害
    * 可設置衝擊力道
    * 可設置觸發核彈衝擊的高撲距離門檻

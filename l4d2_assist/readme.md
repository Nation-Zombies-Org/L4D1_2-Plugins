# 中文說明
特感死亡時顯示人類造成的傷害統計

* 原理
	* 記錄人類對特感造成的傷害，當特改死亡時顯示傷害統計
    * Tank死亡時也會有傷害統計

* 功能
	* 可設置在哪些模式能開啟
    * 可設置只打印Tank的傷害統計

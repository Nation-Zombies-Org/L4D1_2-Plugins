# 中文說明
Tank頭上顯示血量狀態

* 原理
    * 在Tank頭上顯示HP血量狀態
        * 綠色: 血高於一半
        * 橘色: 血低於一半
        * 紅色: 快死掉
        * R.I.P.: 已經死亡
    * 被障礙物或牆壁擋住的時候看不到Tank頭上的血量狀態
    * 只有在tank受傷的時候才會顯示一段時間

# 中文說明
沒有團隊聊天頻道只有公開聊天頻道

* 原理
	* 不管在哪一個頻道說話，全部的人都能看到

* 用意在哪?
    * 不說悄悄話
    * 比較方便，全體玩家打屁哈啦聊天

* 功能
    * 有 !、/、@ 特殊符號開頭的文字訊息不會公開
        * 譬如!admin、/nextmap、@hello

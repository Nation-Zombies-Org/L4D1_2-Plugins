# 中文說明
使用指令顯示人類目前的路程

* 原理
	* 地圖從起點安全室到終點安全室之間路程百分比
		* 0%代表人類目前在起點安全室
		* 95%代表人類快要到終點安全室
	* 最後一關則是從起點安全室到救援區域之間路程百分比
		* 0%代表人類目前在起點安全室
		* 95%代表人類快要到救援區域

* 功能
    * 輸入```!cur```顯示路程

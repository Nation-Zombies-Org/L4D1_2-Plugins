# 中文說明
修復Source引擎的踩頭跳躍bug

* 原理
	* 嘗試修復經典的Source引擎bug，當玩家踩在其他的物件上有機率會造成各種bug，譬如
        * 當玩家踩在殭屍頭上，有一定機率飛到空中墬樓死亡
        * 當玩家踩在Witch頭上，有一定機率直接死亡
        * 踩在榴彈發射器發射的榴彈上高空跳躍
        * 踩在Spitter吐出的酸液物上高空跳躍
        * 踩在隊友扔出去的燃燒瓶、土製炸彈、膽汁瓶上高空跳躍
    * Tickrate越高的伺服器越容易發生

* 功能
	* 無
